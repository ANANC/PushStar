插件名称：EasyScrollView
来源：西山居技术中心自研
插件耦合级别：0

说明：解决需要创建大量对象导致性能问题，配合ScrollView使用。基于UGUI的插件。
 
*耦合级别定义
0-与工程零耦合，可以完全独立运行
1-与框架代码耦合，需要配合框架运行
2-与业务逻辑代码耦合，针对具体功能开发的插件
